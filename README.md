# CarFleet

A TypeScript-based server application for managing car fleet operations.

## Overview

CarFleet is a Node.js and TypeScript project designed to provide a robust backend server for car fleet management. The project uses modern TypeScript practices with proper tooling for development and production builds.

## Features

- Built with **TypeScript** for type-safe development
- **Node.js** based server runtime
- Development and production build support
- Easy-to-use command-line interface for running the server

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

## Usage

### Development Mode

Run the server in development mode with ts-node:

```bash
npm run dev
```

### Production Build

Build the TypeScript code to JavaScript:

```bash
npm run build
```

Run the compiled application:

```bash
npm start
```
